Obrigado por escolher Meu App! Estamos entusiasmados por ter você como parte de nossa comunidade. Nosso aplicativo foi desenvolvido para ajudar você a se organizar e atingir seus objetivos acadêmicos.

Esse app é um projeto de faculdade para testar:

  • Criação de menu
  • Navegação dentro da aplicação
  • Implementar componentes
  • Criar fluxo e logica dentro da app

Breve tour do aplicativo:

  • Login/Boas vindas: Apresenta uma pagina de login com simulação de username e password que direciona para esta pagina de boas vindas
  • Home: Apresenta uma lista que apresenta uma mensagem toast
  • Informe Dados: Apresenta um botão para clicar e usar o component alert com inputs
  • Badges: Uma pagina com texto a minha escolha para apresentar o uso do component badge
